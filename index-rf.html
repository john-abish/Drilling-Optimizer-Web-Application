Randon Forest Regression

A random forest regressor was used to model the ROP function from the drilling data. The regressor used the random forest algorithm to calculate a regression function which depends on WOB, N and Q. A fixed number of decision trees were created which established a relationship between the input variables and the desired output. The algorithm made use of supervised learning and worked as a predictor for regression analysis. The algorithm also had various hyperparameters, including the number of estimators, maximum number of features, and the maximum depth of the tree. These hyperparameters were selected through cross validation of the function. The randomized search cross-validation method was used to obtain the optimal hyperparameter values. After cross validation, the number of estimators was set to 400, the maximum number of features to two, and the maximum depth to ten.
